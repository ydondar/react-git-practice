## React Todo App

Simple Todo application built with React.

### Features

* Add new todos
* Delete todos
* Edit existing todos
* Prevent duplicate entries
* Mark as complete / undo
* Filter todos (All / Active / Completed)
* Persistent storage with LocalStorage
* Auto-focus input while editing

### Tech Stack

* React
* JavaScript (ES6+)
* useState
* useEffect
* useRef
* LocalStorage API
